# 🚗 Car Rental Analytics System  
## Project Charter

---

## 1. Project Background

Many car rental companies lack centralized visibility into:
- Fleet utilization
- Revenue trends
- Customer rental behavior
- Late return impact

Manual tracking and fragmented reporting reduce operational efficiency and strategic planning capabilities.

---

## 2. Project Purpose

To design and implement a structured relational database and analytics dashboard that supports data-driven decision-making for rental operations.

---

## 3. Project Objectives

- Design normalized relational database (BCNF)
- Implement business logic using SQL
- Develop KPI-driven analytics queries
- Build interactive Power BI dashboard
- Document system architecture and business rules

---

## 4. Project Scope

### Included
- Vehicle, Customer, Rental, Finance schemas
- Revenue & utilization KPI analysis
- Dashboard visualization
- Sample dataset

### Excluded
- Real-time production deployment
- Payment gateway integration
- Customer-facing web application

---

## 5. Key Deliverables

- ER Diagram
- Data Dictionary
- SQL Scripts
- KPI Analysis Queries
- Power BI Dashboard
- Executive Summary Report

---

## 6. Risks & Assumptions

### Risks
- Limited real-world data availability
- Scope expansion beyond analytics

### Assumptions
- Business rules remain consistent
- Dataset represents realistic rental activity

---

## 7. Success Criteria

- Database integrity maintained
- KPIs accurately calculated
- Dashboard supports executive-level decision-making
